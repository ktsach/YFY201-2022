
Το εν λόγω πρόγραμμα python3 (spont_decay_v2.py) αποτελεί Monte-Carlo προσομοίωση του μηχανισμού της αυθόρμητης ραδιενεργού διάσπασης.

Aποτελεί τροποποιημένη και εμπλουτισμένη με νέα προγραμματιστικά στοιχεία, γραφήματα και υπολογιστικά αποτελέσματα μορφή του αυθεντικού προγράμματος (decaysound.py / listing #6.6 (p.282) of 'Computational Problems for Physics' book) 

(αναλυτικότερα: βλ. ktsachalinas.spont_decay_report_analytical.pdf)

Αρχεία που συμπεριλαμβάνονται στο παρόν github directory (YFY201-2022):

k.tsachalinas.spont_decay_v2.python-code.pdf : κώδικας σε αρχείο pdf
k.tsachalinas.spont_decay_report_analytical : αναλυτικό report της εργασίας
